Everglow Design – Egen Hjemmeside

Kunde

Everglow Design (eget projekt)

Opgave

At udvikle en professionel, hurtig og brugervenlig hjemmeside fra bunden, der præsenterer ydelser, cases og kontaktmuligheder – uden brug af tunge CMS-systemer.

Målet var fuld kontrol over kode, performance og design samt en løsning, der kan hostes gratis og nemt vedligeholdes.

Løsning

Hjemmesiden er udviklet i Visual Studio Code med fokus på ren og struktureret frontend-kode.

Der er arbejdet med:

  • Klar informationsstruktur
  • Enkel navigation
  • Fokus på brugervenlighed og læsbarhed
  • Responsivt design, der fungerer på både desktop og mobil

Siden hostes via GitHub Pages, hvilket giver en stabil og omkostningsfri løsning.

Teknologier

  • HTML5
  • CSS3
  • JavaScript
  • Visual Studio Code
  • GitHub Pages

Resultat

Resultatet er en let, hurtig og overskuelig hjemmeside, der:

  • Loader hurtigt
  • Er nem at navigere i
  • Fremstår professionel og troværdig
  • Kan udvides løbende med cases og blogindhold

Projektet fungerer samtidig som en praktisk demonstration af mine kompetencer inden for webintegration og frontend-udvikling.

Læring og Værdi

Casen viser min tilgang til:

  • Struktur og semantisk HTML
  • Vedligeholdelsesvenlig kode
  • Brug af Git og versionsstyring
  • Opsætning af statiske websites uden unødige omkostninger
